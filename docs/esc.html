ESC設定

ESC設定

ESC（Electric Speed Controller）はブラシレスモータの回転数を制御する電子デバイスである．ESCにはRacerstar RS30A V2を使用する．

設定ソフト：BLHeliSuite

Make Interfaces

  • Arduino Board: Nano w/ ATmega328(UNOを使用)
  • Baud: 115200
  • Arduino BLHeli Bootloader

Pinouts Arduino UNO ESC 　GND-------GND
D3--------Signal

Select ATMEL/ SILABS Interface -> C SILABS Bootloader(USB/Com)

Select Port -> Connect -> Read Setup
